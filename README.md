# Autobiography & Portfolio (Streamlit)

This is a small Streamlit application that serves as a personal autobiography and portfolio template. It demonstrates:

- About / Quick facts
- Portfolio projects with images and links
- A small project list
- A tiny data visualization gallery using Altair
- Contact form (demo-only)
- Resume download (TXT)

## Requirements

Install dependencies (recommended in a virtualenv):

```powershell
python -m pip install -r requirements.txt
```

## Run

```powershell
streamlit run "d:\\College Subjects Folder\\3Y1S\\CS365 (Data Analytics and Visualization)\\app.py"
```

Replace the placeholder name, image, and links with your own content.
